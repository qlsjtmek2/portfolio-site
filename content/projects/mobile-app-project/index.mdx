# 모바일 앱 프로젝트

React Native를 활용하여 iOS와 Android 모두에서 동작하는 크로스 플랫폼 모바일 애플리케이션을 개발했습니다.

## 프로젝트 소개

사용자의 일상을 더욱 편리하게 만들어주는 생산성 향상 애플리케이션입니다.

## 핵심 기능

- **오프라인 지원**: 인터넷 연결 없이도 핵심 기능 사용 가능
- **푸시 알림**: 중요한 이벤트에 대한 실시간 알림
- **다크 모드**: 눈의 피로를 줄이는 다크 테마 지원
- **데이터 동기화**: 클라우드를 통한 멀티 디바이스 데이터 동기화

## 사용 기술

- React Native
- TypeScript
- Redux Toolkit
- Firebase
- AsyncStorage

## 프로젝트 성과

- App Store 평점 4.7/5.0
- Google Play 다운로드 50,000+
- 일일 활성 사용자 5,000명

## 도전 과제

### 성능 최적화
React Native의 특성상 네이티브 앱과 동일한 성능을 내기 위해 많은 최적화 작업이 필요했습니다.

### 플랫폼별 대응
iOS와 Android의 디자인 가이드라인 차이를 고려한 UI/UX 구현이 필요했습니다.

## 향후 계획

- 위젯 기능 추가
- Apple Watch, Wear OS 지원
- AI 기반 추천 기능 도입
