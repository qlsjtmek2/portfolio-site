## Retro Runner

Phaser 3와 TypeScript로 제작한 80년대 아케이드 감성의 레트로 러너 게임입니다. 네온 무지개 테마와 향수를 자극하는 비주얼로 클래식 게임의 즐거움을 재현했습니다.

🎮 **[게임 플레이하기](https://qlsjtmek2.github.io/retro-runner/)**

### 프로젝트 개요

80년대 아케이드 게임의 감성을 현대적인 기술로 재해석한 러너 게임입니다. 심플한 조작과 중독성 있는 게임플레이로 누구나 쉽게 즐길 수 있습니다.

### 주요 기능

#### 게임플레이
- **이중 점프 시스템**: 최대 2번까지 공중 점프 가능
- **빠른 낙하**: 방향키로 빠르게 낙하하여 장애물 회피
- **프로그레시브 난이도**: 시간이 지날수록 속도와 장애물 생성 빈도 증가
- **3가지 장애물 높이**: Low, Medium, High 장애물로 다양한 패턴 구성

#### 비주얼 & 효과
- **네온 무지개 테마**: 80년대 아케이드 감성의 네온 색상
- **Squash & Stretch 애니메이션**: 부드러운 캐릭터 움직임
- **파티클 효과**: 착지와 충돌 시 파티클 효과
- **레트로 컬러 팔레트**:
  - 배경: 딥 퍼플 (#0f0f23)
  - 플레이어: 네온 옐로우 (#FFFF00)
  - 장애물: 네온 퍼플 (#A855F7)
  - UI: 시안 (#00ffff)

### 개발 하이라이트

#### 1. 물리 기반 게임플레이
Phaser 3의 Arcade Physics를 활용하여 자연스러운 점프와 중력을 구현했습니다.

```typescript
// 이중 점프 구현
if (jumpCount < 2) {
  player.setVelocityY(-500);
  jumpCount++;
}
```

#### 2. 애니메이션 시스템
Squash & Stretch 애니메이션으로 캐릭터에 생동감을 부여했습니다.

- 점프 시: 세로로 늘어남
- 착지 시: 납작해짐
- 이동 중: 자연스러운 변형

#### 3. 프로그레시브 난이도
게임 시간에 따라 난이도가 점진적으로 상승합니다.

```typescript
// 시간에 따른 속도 증가
gameSpeed = baseSpeed + (score * 0.1);
```

#### 4. 파티클 효과
착지와 충돌 시 네온 파티클 효과로 시각적 피드백을 제공합니다.

### 게임 컨트롤

- **스페이스바 / 위 방향키**: 점프 (최대 2회)
- **아래 방향키**: 빠른 낙하
- **자동 진행**: 캐릭터는 자동으로 앞으로 이동

#### 80년대 아케이드 감성
네온 컬러와 픽셀 그래픽으로 80년대 아케이드 게임의 향수를 재현했습니다.

#### 미니멀리즘
심플한 비주얼과 직관적인 UI로 게임플레이에 집중할 수 있도록 했습니다.

밝고 화려한 네온 컬러로 역동적이고 흥미진진한 분위기를 연출했습니다.

### 개발 과정

#### 초기 프로토타입
Phaser 3로 기본적인 러너 게임 메커니즘을 구현했습니다. 플레이어 이동, 점프, 장애물 생성 등 핵심 기능을 먼저 완성했습니다.

#### 게임플레이 개선
이중 점프와 빠른 낙하 기능을 추가하여 게임플레이에 깊이를 더했습니다. 다양한 장애물 높이로 패턴을 복잡하게 만들었습니다.

#### 비주얼 폴리싱
Squash & Stretch 애니메이션과 파티클 효과를 추가하여 게임에 생동감을 불어넣었습니다. 네온 컬러 팔레트로 80년대 감성을 완성했습니다.

#### 난이도 조절
테스트를 통해 적절한 난이도 곡선을 찾아냈습니다. 초반에는 쉽게 시작하지만 점차 도전적으로 변하도록 조정했습니다.

### 배운 점

Phaser 3 게임 엔진의 강력함을 체감했습니다. 물리 엔진, 애니메이션 시스템, 파티클 효과 등 게임 개발에 필요한 모든 것이 잘 갖춰져 있었습니다.

특히 TypeScript를 사용하여 타입 안정성을 확보하면서도 빠르게 개발할 수 있었습니다. 게임 로직이 복잡해질수록 TypeScript의 장점이 더욱 부각되었습니다.

게임 디자인에서 가장 중요한 것은 플레이어 피드백임을 깨달았습니다. 애니메이션, 파티클 효과, 사운드 등 모든 요소가 플레이어에게 즉각적인 반응을 제공해야 합니다.
